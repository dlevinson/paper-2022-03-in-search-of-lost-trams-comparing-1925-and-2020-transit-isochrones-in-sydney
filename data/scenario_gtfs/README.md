# Scenario GTFS Files

This directory contains the paper-specific GTFS scenario bundles used for the six published scenarios. The baseline historical Sydney GTFS source is already public at https://hdl.handle.net/2123/34748 and https://doi.org/10.25910/07z3-bn41; this package keeps only the scenario bundles needed to identify the 2022 Findings paper setup.

S3 and S6 include pointer files where the source bundle is byte-identical to an already packaged S1 or S2 bundle.
